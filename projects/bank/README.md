# Bank Project

Simple banking application.

## Install

Make sure you have G++ installed in your machine.

```sh
  git@github.com:cyruzin/cpp-study.git

  cd projects/bank/src/
```

## Compile

```sh
  g++ main.cpp ./lib/*.cpp -o main
```

## Run

Windows:

```sh
  ./main.exe
```

Linux:

```sh
  ./main
```
